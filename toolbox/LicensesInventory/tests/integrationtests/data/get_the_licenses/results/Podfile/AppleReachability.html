Installation Guide for AppleReachability

You want to add pod 'AppleReachability', '~> 5.0' similar to the following to your Podfile:

target 'MyApp' do
  pod 'AppleReachability', '~> 5.0'
end

Then run a pod install inside your terminal, or from CocoaPods.app.

Alternatively to give it a test run, run the command:

pod try AppleReachability
